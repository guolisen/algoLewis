# std::lower_bound  std::upper_bound

ForwardIter lower_bound(ForwardIter first, ForwardIter last,const _Tp& val)算法返回一个非递减序列[first, last)中的第一个大于等于值val的位置。

ForwardIter upper_bound(ForwardIter first, ForwardIter last, const _Tp& val)算法返回一个非递减序列[first, last)中第一个大于val的位置。
```
// lower_bound/upper_bound example
#include <iostream>     // std::cout
#include <algorithm>    // std::lower_bound, std::upper_bound, std::sort
#include <vector>       // std::vector

int main () {
  int myints[] = {10,20,30,30,20,10,10,20};
  std::vector<int> v(myints,myints+8);           // 10 20 30 30 20 10 10 20

  std::sort (v.begin(), v.end());                // 10 10 10 20 20 20 30 30

  std::vector<int>::iterator low,up;
  low=std::lower_bound (v.begin(), v.end(), 20); //          ^
  up= std::upper_bound (v.begin(), v.end(), 20); //                   ^

  std::cout << "lower_bound at position " << (low- v.begin()) << '\n';
  std::cout << "upper_bound at position " << (up - v.begin()) << '\n';

  return 0;
}
```
```
lower_bound at position 3
upper_bound at position 6

# std::tuple
```
// tuple example
#include <iostream>     // std::cout
#include <tuple>        // std::tuple, std::get, std::tie, std::ignore

int main ()
{
  std::tuple<int,char> foo (10,'x');
  auto bar = std::make_tuple ("test", 3.1, 14, 'y');

  std::get<2>(bar) = 100;                                    // access element

  int myint; char mychar;

  std::tie (myint, mychar) = foo;                            // unpack elements
  std::tie (std::ignore, std::ignore, myint, mychar) = bar;  // unpack (with ignore)

  mychar = std::get<3>(bar);

  std::get<0>(foo) = std::get<2>(bar);
  std::get<1>(foo) = mychar;

  std::cout << "foo contains: ";
  std::cout << std::get<0>(foo) << ' ';
  std::cout << std::get<1>(foo) << '\n';

  return 0;
}
```

1. **创建元组** **foo** **和** **bar**：

- std::tuple<int,char> foo (10,'x');

创建一个包含 int 和 char 的元组 foo，初始值为 (10, 'x')。

- auto bar = std::make_tuple ("test", 3.1, 14, 'y');

通过 std::make_tuple 创建元组 bar，包含字符串字面量、double、int、char，初始值为 ("test", 3.1, 14, 'y')（索引从 0 开始）。

1. **修改** **bar** **的元素**：

- std::get<2>(bar) = 100;

访问 bar 的第 2 个元素（索引为 2），将其从 14 修改为 100。此时 bar 的值为 ("test", 3.1, 100, 'y')。

1. **解包元组到变量（****std::tie****）**：

- std::tie (myint, mychar) = foo;

将 foo 的元素解包到 myint 和 mychar。此时 myint = 10，mychar = 'x'。

- std::tie (std::ignore, std::ignore, myint, mychar) = bar;

解包 bar 的元素，忽略前两个元素（用 std::ignore），将第 2 个元素（100）赋给 myint，第 3 个元素（'y'）赋给 mychar。此时 myint = 100，mychar = 'y'。

1. **再次修改** **mychar**：

- mychar = std::get<3>(bar);

bar 的第 3 个元素是 'y'，因此 mychar 仍为 'y'（无变化）。

1. **修改** **foo** **的元素**：

- std::get<0>(foo) = std::get<2>(bar);

bar 的第 2 个元素是 100，因此 foo 的第 0 个元素从 10 改为 100。

- std::get<1>(foo) = mychar;

mychar 的值是 'y'，因此 foo 的第 1 个元素从 'x' 改为 'y'。

此时 foo 的值为 (100, 'y')。

1. **输出** **foo** **的元素**：

- 打印 std::get<0>(foo)（100）和 std::get<1>(foo)（'y'）。

# std::remove

```
template <class ForwardIterator, class T>
  ForwardIterator remove (ForwardIterator first, ForwardIterator last, const T& val)
{
  ForwardIterator result = first;
  while (first!=last) {
    if (!(*first == val)) {
      if (result!=first)
        *result = move(*first);
      ++result;
    }
    ++first;
  }
  return result;
}
```

```
template <class ForwardIterator>
  ForwardIterator unique (ForwardIterator first, ForwardIterator last)
{
  if (first==last) return last;

  ForwardIterator result = first;
  while (++first != last)
  {
    if (!(*result == *first))  // or: if (!pred(*result,*first)) for version (2)
      *(++result)=*first;
  }
  return ++result;
}

```
template <class InputIterator, class T>
   T accumulate (InputIterator first, InputIterator last, T init)
{
  while (first!=last) {
    init = init + *first;  // or: init=binary_op(init,*first) for the binary_op version
    ++first;
  }
  return init;
}

```
template <class InputIterator, class T>
  typename iterator_traits<InputIterator>::difference_type
    count (InputIterator first, InputIterator last, const T& val)
{
  typename iterator_traits<InputIterator>::difference_type ret = 0;
  while (first!=last) {
    if (*first == val) ++ret;
    ++first;
  }
  return ret;
}
